# Essential Commands

Day 1

    git clone <url>

    git status

    git add <file>

    git commit -m "msg"

    git push <remote> <branch>

    git pull <remote> <branch>

    git log --oneline

    git log --oneline --graph --all

Day 2

    git checkout -b <new-branch>

    git push -u origin <branch>

    git diff

    git diff --staged

    git stash

    git stash pop

    git log --oneline --graph --all

    git branch -d <branch>

    git pull origin --delete <branch>

Day 3

    git reset --soft <commit>             #Local Only

    git reset --mixed <commit>            #Local Only

    git reset --hard <commit>             #DANGER

    git revert <commit>                   #SAFE

    git rebase <base-branch>              #Local Only

    git reflog                            #Info Only

    git cherry-pick <commit>              #Contextual
